\CWProblem{

Линеаризовать циклическую строку, то есть найти минимальный в лексикографическом смысле разрез циклической строки.\newline

\textbf{Формат ввода}\newline
Некий разрез циклической строки.\newline

\textbf{Формат вывода}\newline 
Минимальный в лексикографическом смысле разрез.\newline
}
